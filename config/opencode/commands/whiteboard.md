Create a text-based architecture diagram (ASCII/boxes) showing how the components in this project interact. Include:
- Entry points (API routes, main functions, UI components)
- Core modules and their responsibilities
- Data flow (what calls what, what passes where)
- External dependencies (databases, APIs, services)
- Boundaries between systems
